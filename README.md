## Run Locally

Clone the project

```bash
  git clone https://github.com/apvarun/contributors-table-generator.git
```

Go to the project directory

```bash
  cd contributors-table-generator
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```
